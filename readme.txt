Customize woocommerce products.

== Description ==

Plugin to customize the vision of woocommerce products.

It has a simple and easy-to-use setup.

This plugin for WooCommerce provides the following functionalities:
- Visual configuration of the individual page of each product and the product loop.
- Add additional content in the individual product view and in the product loop.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
